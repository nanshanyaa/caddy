项目最终需求（保持不变）
- 面板用于管理多台服务器上的 Docker 容器，并控制公网 VPS 上的 Caddy 进行反向代理。
- 通过 Caddy API（2019）下发反代规则；下线时删除规则。
- Agent 上报心跳与容器状态；删除源站触发自毁。
- UI 支持自定义 Logo/背景、容器图标；侧边栏分区（线路/源站）。
- 解决 Caddy 启动失败，确保 2019 端口可用并完成联调。

修改记录（简洁）
- backend/app/main.py: 发布时若目标为内网 IP 且源站有 public_ip，自动用 public_ip 作为反代目标；下线时使用线路自定义 apiPort。
- backend/app/main.py: 删除源站后进入短期“待杀”队列，心跳返回 kill，避免源站自动恢复（不使用黑名单）。
